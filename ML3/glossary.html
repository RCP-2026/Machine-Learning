Machine Learning 3 - Glossary

Glossary ML3
Gaussian Mixture Model (GMM): A probabilistic clustering model that represents data as a weighted combination of multiple Gaussian distributions, allowing clusters to overlap and be described in terms of likelihood.
Mixture Component: An individual Gaussian distribution within a Gaussian Mixture Model, defined by its own mean, covariance, and relative weight.
Component Weight: The proportion of the overall dataset attributed to a specific mixture component, representing how much that component contributes to the model.
Covariance Matrix: A matrix that describes the variance of each feature and the correlation between features within a Gaussian component, controlling the shape and orientation of the distribution.
Soft Assignment: A clustering outcome in which each data point is assigned probabilities of belonging to each mixture component rather than a single hard label.
Membership Probability: The probability that a data point belongs to a given mixture component, as returned by a fitted Gaussian Mixture Model.
Synthetic Clustered Data: Artificially generated data with known structure, used to explore and visualise how clustering models behave under controlled conditions.
Adjusted Rand Index: A numerical measure of similarity between two cluster labelings that accounts for agreement occurring by chance, used to compare predicted clusters with known groupings.
NIfTI Image: A standard file format for storing medical imaging data, commonly used for three-dimensional and four-dimensional volumetric scans.
Voxel: A volumetric pixel representing a single intensity value at a specific spatial location in a three-dimensional medical image.
Image Modality: A specific type of medical imaging scan where the same anatomy is captured using different acquisition settings, resulting in different intensity characteristics.
Intensity Distribution: The spread and frequency of voxel intensity values within an image or region of interest.
Mask: A boolean array used to include or exclude specific voxels from analysis, often applied to remove background regions from medical images.
Intensity-Based Segmentation: A segmentation approach that groups voxels based solely on their intensity values, without using spatial or anatomical information.
Label Map: A three-dimensional array in which each voxel stores a cluster label, allowing segmentation results to be visualised in spatial context.
Multi-modal Clustering: Clustering performed using multiple image modalities simultaneously, treating each modality’s voxel intensity as a separate feature.
